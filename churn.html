featured project

project CHURNGUARD

Smart CRM Analytics System designed to predict customer churn, automate retention strategies, and analyze sentiment for digital subscription services.

• Project Overview

ChurnGuard is a machine-learning-based decision support system developed to help streaming platforms tackle subscriber churn. By implementing highly interpretable Logistic Regression models optimized with Class Weight techniques, ChurnGuard identifies at-risk subscribers using usage and billing data. It seamlessly bridges data science with business operations by triggering automated personalized email interventions, protecting company revenue and improving retention.

• Core Features

Equipped with analytical intelligence and automated intervention logic, the platform operates without requiring manual assessment:

  • Customer Risk Prediction: Leverages ML classification to evaluate customer risk categories based on account age, viewing hours, monthly charges, and support tickets.
  • Automated Intervention & Recommendations: Instantly generates tailored retention strategies and dispatches them directly via automated email triggers.
  • Sentiment Analysis Search Engine: Employs a custom-trained Multiclass Support Vector Machine (SVM) to classify and visualize user reviews (Positive, Neutral, Negative).
  • Visual Dashboard: Displays real-time churn dynamics, user distributions, and engagement analytics in an elegant, SaaS-style web interface.

• Language & Core System Environment

The system is built on a highly integrated full-stack and ML pipeline:

REACT Tailwind NodeJS & Express MySQL Python & FastAPI Scikit-Learn Resend gcp Figma Git

• Generated System Outputs

Upon analyzing customer profiles and public text data, the platform issues:

  • Personalized Strategy Emails: Automatic promotional/intervention emails sent via SMTP to at-risk clients.
  • CRM Predictive Reports: Exportable CSV datasets detailing churn risks and calculated customer value metrics.
