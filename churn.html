project BEBASPUSTAKA

Sistem Informasi Bebas Pustaka dirancang untuk mendigitalisasi dan mengotomatisasi kompensasi (kompen) mahasiswa Jurusan TI

• Project Overview

Bebas Pustaka website was developed to digitalize and automate the the event of kompensasi(kompen) tasks within the IT Department, which previously relied on inefficient manual procedures. By integrating APIs between the library database (OPAC) and the SIKOMPEN platform, this Laravel-based system eliminates the need for physical queues and slashes book-loan verification times to just one second. Consequently, it reduces the administrative workload for librarians during peak end-of-semester periods by over 90%.

• Core Features

Sistem ini ditanamkan logika automasi untuk mengecek status pinjaman buku mahasiswa secara mandiri tanpa intervensi manual:

  • dashboard: Mengvisualisasi data database peminjaman buku perpustakaan yand dapat di filterisasi sesuai keingignan user
  • filterisasi bebaspustaka kompen:proses otomatis yang dilakukan oleh sistem ketika memulai proses approval dan melakukan filterisasi mahasiswa yang sudah bebaspustaka sesuai syarat yang sudah ditentukan.
  • bebasputaka kompen approval:proses approval untuk semua mahasiswa yang sudah memenuhi syarat approval.
